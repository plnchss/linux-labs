#  Linux & Bash Automation Labs

Архив лабораторных работ по администрированию Linux. Основной упор сделан на автоматизацию рутинных задач с помощью Bash-скриптов.

---

##  Лабораторная работа №1: Системный монитор
**Задача:** Написать интерактивный скрипт для сбора информации о системе.

* **Функционал:** Сбор данных (каталоги, процессы, ОС, пользователи, диски, память, пакеты).
* **Режимы:** Интерактивное меню или автоматический вывод в файл через аргумент `--tofile`.
* **Интерфейс:** Русский язык, постраничный вывод с паузой через `Enter`.

---

##  Лабораторная работа №2: Bash Automation
Набор скриптов для углубленной работы с файловой системой и процессами:

1. **Шахматная доска:** Генерация доски любого размера с использованием ANSI-цветов (без нахождения остатка от деления).
2. **Аналог `du`:** Рекурсивный подсчёт размера папок с помощью `find` и `stat` (без использования `du` и `ls`).
3. **Сортировщик файлов:** Автоматическая сортировка файлов по расширениям в подпапки.
4. **Backup с ротацией:** Архивация (`tar.gz`) директорий с очисткой архивов старше 7 дней.
5. **Анализ текста:** Подсчет топ-N часто встречающихся слов в файлах с нормализацией текста и игнорированием знаков препинания.

---

## Ключевые навыки
* **Bash Scripting:** Ассоциативные массивы, регулярные выражения, работа с потоками.
* **Системное администрирование:** Управление процессами, пользователями и дисковым пространством.
* **Алгоритмизация:** Решение сложных задач без стандартных высокоуровневых команд.
